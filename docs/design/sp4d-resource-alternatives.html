Resources — four layout alternatives

All four are built from the sheet's real materials — Libre Baskerville / Noto Sans, the crimson-underline headings, the archivist-toggle-box pip, the − 6/9 + counter, plain reset labels, and the crimson Recover button. Only the layout differs. Each renders the same character: Rage (2/3), Bardic Inspiration d8 (3/4), Sorcery Points (6/9), Arcane Recovery (available). The current shipped design is the plain badge strip; A–D are the alternatives.

AUnified table — “the ledger”

Resources as rows in the same table the Features tab already uses (dashed row rules, serif names, source sub-line). Densest and most scannable; the natural choice for a high-level character with many pools. Pool · Reset · Action columns mirror the Features table exactly.

The current badge strip, but the badges are grouped under the sheet's block sub-headings by when they recharge. Answers the question you actually ask at a rest — “what comes back on a short rest vs a long rest?” — without changing the badge itself.
