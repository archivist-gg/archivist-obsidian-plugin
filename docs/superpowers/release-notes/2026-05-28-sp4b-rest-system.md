### SP4b — Rest System

- Two new buttons in the hero (☾ Short Rest / ★ Long Rest) sit between the
  identity and the AC shield.
- Long rest opens an opt-out modal: HP to max, hit-dice regained
  (most-spent-first), spell slots, exhaustion, feature uses, and magic-item
  charges each get a row. Untick any to skip; Confirm commits.
- Short rest adds a hit-dice spend strip — pip toggles select dice, then
  ⚄ Roll & Apply, Apply Avg, or ✎ Manual commits the heal. The roll log
  records each spend (random / avg / manual).
- Concentration clears unconditionally on long rest (RAW). Death saves
  clear automatically when HP returns from 0.
- Pact slots (warlock) and homebrew rest variants remain deferred to SP4c.
